Nachfahrenselektor vs. Kindselektor

Aufgabe: Die erste Listenebene soll eingerückt werden, die zweite nicht.

Der Nachfahrenselektor

Mit nav ul { padding-left: 0; } werden beide Listenebenen eingerückt.

  • Startseite
  • Reisen
    • Italien
    • Island
  • Kontakt

Der Kindselektor

Mit nav > ul { padding-left: 0; } wird nur die erste Listenebene eingerückt.

  • Startseite
  • Reisen
    • Italien
    • Island
  • Kontakt
