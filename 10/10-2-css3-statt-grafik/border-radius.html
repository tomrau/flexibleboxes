Runde Ecken per CSS3: border-radius

Beispiel Nr. 1

Diese Box hat gleichmäßig abgerundete Ecken.

div { 
  background: #a8caf3; 
  width: 25em; 
  padding: 0.5em; 
  border: 0.25em solid #175ba2; 
  border-radius: 0.5em; 
}

Beispiel Nr. 2

div {

  width: 25em; 
  padding: 0.5em;

Beispiel Nr. 3

Diese Box hat verschiedene abgerundete Ecken.

div { 
  background: #a8caf3; 
  width: 25em; 
  padding: 0.5em; 
  border-radius: 0.25em 3em 0.25em 3em; 
}
